Sunnyside School

Overview

This project aims to redesign the website for the Sunnyside elementary, which is a non-profit school, and launch it before the school year starts. This project is done by a teamwork with the director Natasha Podogova and the volunteers from Sunnyside.
Visit the live site

Problems and changes

Navigation

On the old site, it shows the main menu under school logo, which is accessible on every page, but the sub menus are only illustrated on the relative main pages. There are a few problems in this navigation. First of all, the user has to check each page listed on the main menu to view the subpages. And it is very easy to get lost from one subpage to another. Also it very confusing that some main pages share same submenus. The user needs to take guesses to find the wanted page.

According to the problems shown in the old one, we arranged the navigation in a new way that minimized main pages from seven to four. And put all the submenus under each section, which are easy access to all pages.
